Introduction

Markdown est un laguage de balisage qui permet de formater du text simple. On peut y ajouter des textes gras, du text italic, des liens, des images, etc. La particularité de markdown est que le text reste lisible même sans le rendu.

IL est beaucoup utiliser pour formater les fichier readme.md des dépots github, pour formater les notebook jupiter, etc.

Les titres en Markdown

Markdown utilise le symbole "#" pour faire les titres. Mettez seulement le nombre de dièse correspondant au niveau du titre (Comme en HTML) suivi du titre lui même. Exemple # Ceci est titre de niveau H1
## Ceci est titre de niveau H2
### Ceci est titre de niveau H3
#### Ceci est titre de niveau H4
##### Ceci est titre de niveau H5
###### Ceci est titre de niveau H6

Le gras

Pour mettre du text en **gras** en markdown, il suffit juste de entourer de le placer entrer double étoile ou deux double underscore. Exemple : **Ceci est du text en gras**
__ Ceci aussi __

L italic

Pour mettre du text en *italic* en markdow on le met entre

  • Soit une paire d'étoiles
  • Soit une paire d'underscore
Exemple : _ Cet text en italic _
* Ce text aussi *

Les citations

Markdown offre la possiblité de formater un text un text comme une citation. > IL suffit de mettre le symbole ""> devant le text. Exemple > Ceci est une citation

Les listes non ordonnees

Pour créer des listes non ordonnées en markdown, on utilise

  • Le caractère étoile "*"
  • Le caractère tiret "-"
  • Le caractère plus "+"
- element + element * element

Les listes ordonees

Les ordonné sont contruites en mettant "1."devant chaque élément de la liste quelque soit son ordre. Exemple : 1. premier element
1. deuxième element
1. troisième element

Les blocks de code

Les blocks de code sont intruit dans une ligne en utilisant une paire de "`" pour du code qui se limite à quelque mots et une pair de "```" pour du code multiligne.

Les liens

Les liens en markdown sont ajoutés en utilisant la syntaxe suivante : [Text du lien](https://www.adressedusite.com)

Les images

Les images utilisent la même syntaxe que les liens à la différence qu'il faut ajouter un "!" devant le lien comme suit : [Text alternatif](https://www.liendelimage.com)
